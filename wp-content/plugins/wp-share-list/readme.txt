wp-share-list

== Description ==

<p>在文章末尾生成一个文章分享站点列表（包含大部分流行的分享站点）。</p>
<p>自定义显示位置：<?php if(function_exists('wp_share_list')) wp_share_list(); ?></p>
<h3>功能特性</h3>
<ul>
<li>1、不经过第三方网站的跳转（明河写这个插件的初衷，就是很反感其他插件都带有第三方logo，点击后还要跳转到那网站）</li>
<li>2、ajax加载列表数据，晚加载分享站点图标。（尽量不影响页面加载速度）</li>
<li>3、自由控制分享站点的显示及其排序</li>
<li>4、可控制是否引入jquery库（防止重复加载，毕竟现在很多主题都已经自带jquery了）</li>
</ul>
<p>备注：IE6由于无法显示png图片，所以明河对其效果做了单独处理，不显示阴影</p>
<p>备注：默认是不自动添加分享站点列表的，你想要自动显示，请勾选后台管理页面中的“自动添加到文章尾部”</p>

== Installation ==
<h3>安装</h3>
<ol>
<li>下载并解压
<li>将此文件夹上传到Wordpress的插件目录：'Wordpress根目录/wp-content/plugins/'。</li>
<li>在后台的插件管理处激活。</li>
</ol>
